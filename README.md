# TaskFlow - Premium Task Manager

A modern, beautiful, and fully functional Task Management Web Application built with HTML, CSS, and JavaScript.

## ✨ Features

- ✅ Add, Edit, and Delete Tasks
- ✅ Mark Tasks as Completed or Pending
- ✅ Real-time Search Functionality
- ✅ Smart Filters (All Tasks, Completed, Pending)
- ✅ Task Categories (Work, Personal, Shopping)
- ✅ Priority Levels (Low, Medium, High)
- ✅ Due Date Support
- ✅ Dark Mode Toggle
- ✅ Fully Responsive Design (Mobile + Desktop)
- ✅ Auto-save using LocalStorage (Data persists in browser)
